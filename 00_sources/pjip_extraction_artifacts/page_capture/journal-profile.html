About PJIP Journal Profiles

Every journal indexed in the PJIP has a unique profile page like this one that displays all of the information we have for the journal. These profiles include:
  • Basic Journal Details: Subject, sub-discipline, eISSN, publisher, location, language, and the average number of papers the journal publishes per year.
  • Journal Policies: Key policies on word/page limits, open-access status, peer review anonymity, and accepted paper types (e.g., book reviews, discussions).
  • Ranking Metrics: Prominent rankings such as SJR, SNIP, Impact Factor, CiteScore, Citation Percentage, Self-Citaiton percentage, and H-index.
  • Operational Statistics: Latest stats from the past year, including submission volume, acceptance rate, desk rejection rate, and average response times.

If you notice any incorrect information, please let us know at [email protected].
To help complete missing data, journals can refer to the PJIP’s Indexing Guide.
